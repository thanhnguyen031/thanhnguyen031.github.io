Client Project Overview

Project Identity

Website Name: The Nguyen Collective

Client Name: Thanh Nguyen

Date: March 31, 2025

Project Purpose

This project is a personal branding website designed to professionally showcase Thanh Nguyen’s technical and creative work. It acts as a central portfolio and digital résumé for networking, collaboration, and career development. The design will emphasize clarity, simplicity, responsiveness, and interactivity.

Target Audience

  • Employers and recruiters seeking candidates with web development skills
  • Fellow students and potential project collaborators
  • Creative professionals and developers in related industries
  • Friends, family, and general audiences interested in Thanh’s work

Website Structure

The Nguyen Collective will contain the following main pages:

  • Home: Welcome message, featured projects
  • About: Biography, timeline, downloadable resume
  • Projects: Cards for individual works with tools used
  • Gallery: Filterable and responsive image showcase
  • Contact: Form with validation for inquiries

Site Map

You can also view the sitemap online.

Page-by-Page Details

  • Home: AJAX project loader, introductory message
  • About: Skill progress animation, timeline, headshot
  • Projects: Clickable cards with modal details
  • Gallery: Filter tabs, lightbox viewer
  • Contact: JS form validation with success message

Dynamic Features

  • AJAX-based content loading for projects
  • Modal/lightbox expansion for interactive display
  • Filterable gallery grid with animations
  • Real-time form validation using JavaScript
  • jQuery UI enhancements (tabs, transitions, effects)

Conclusion

The Nguyen Collective is a complete personal brand website project showcasing web development and creative strengths. It combines usability with interactivity, representing Thanh Nguyen’s work in a modern and accessible format.
